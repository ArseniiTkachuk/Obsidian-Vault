## REST API 
Це набір правил для спілкування одної програми з іншою через інтернет 
Використовується в веб розробці де потрібно звязати [[Backend]] та [[Frontend]], також використовують для звязання [[Мікросервісна архітектура|мікросервісної архітектури]] 

Стандартом передавання інформації є [[Json]] файли, хоча Resst API і підтримує формат xml але понад 99% розробників використовують [[Json]]

![[Json#JSON (JavaScript Object Notation)]] 

REST використовує стандартні методи протоколу HTTP для виконання дій із даними:

|**Метод**|**Що він робить**|**Аналогія**|
|---|---|---|
|**GET**|Отримує дані (наприклад, список постів)|Читання|
|**POST**|Створює нові дані (новий пост)|Запис|
|**PUT**|Повністю оновлює існуючі дані|Заміна|
|**PATCH**|Частково оновлює дані (зміна лише заголовка)|Редагування|
|**DELETE**|Видаляє дані|Видалення|
