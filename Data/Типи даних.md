
## Типи даних в програмуванні
Типи даних використовують для кращого контролю над памятю, та для швидшого виконання програм.

Мови програмування бувають зі строгою типізацією та з динамічною типізацією. У першому випадку мова програмування дає можливість створювати більш швидкі застосунки в обмін на те що програміст буде змушений керувати памятю в ручну, такими мовами є [[C++]], [[C sharp|C#]] та інші . У другому ж випадку програміст не змушений задумуватися над типом даних але програма буде повільніша адже замість нього це робить компютер угадуючи тип даних, такими мовами є [[Python]], [[Js]].

## Види типів даних
Типи даних можуть відрізнятися від мови до мови але є загальна кількість тих типів які є у більшості мовах програмування:

- Integer (int) => цей тип даних використовується для зберігання цілих чисел та займає 4 байти в оперативній памяті
- float => використовується для зберігання чисел із комою
- bool => використовується для зберігання булевих значень (true або false)
- char => використовується для зберігання одного символа.
