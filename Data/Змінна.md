### Змінні це слова чи слово сполучення в які кладеться якась інформація 
Змінні бувають різних типів та відрізняються способом свого використання залежно від мови в якій ти використовуєш їх 

Щоб краще зрозуміти змінні можна провести аналогію з коробкою, де змінна це коробка а інформація це те що ми кладемо у середину, коробка може бути пустою як і змінна, це значення називається null

##### Приклад на [[Python]]:
```Python
a=0
```
У цьому прикладі я оголосив змінну (а) та вказав їй значення 0 

Змінні мають типи даних що допомагає програмі розуміти скільки місця потрібно виділяти для них 

##### Приклад на [[C sharp|C#]]:
``` C#
using System;

public class Program
{
    public static void Main()
    {
	    // змінна типу int (Integer), 4 байти 
	    int a;
	    
	    // змінна типу long, 8 байт 
	    long b;
    }
}
```

Тут оголошено дві змінні різного типу, ці обидві змінні мають один тип а саме числовий але різниця між ними в тому що одна займає більше а друга менше памяті але та що займає більше памяті може вмістити в себе більше число 

Змінні можуть приймати в себе не тільки числа а й інші типи такі як рядки, булеве значення, числа з комою та інші [[Типи даних]]
