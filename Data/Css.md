CSS (Cascading Style Sheets) — це «стиліст» інтернету. Якщо [[Html]] — це каркас будинку (стіни, двері, вікна), то CSS — це дизайн: колір шпалер, розмір меблів та те, де саме стоятиме диван.

CSS працює за принципом «вибрати елемент і змінити його властивість».

Його структура дуже логічна:
- Селектор (Selector): Назва елемента [[Html]], який ми хочемо змінити (наприклад, h1 або p).
- Властивість (Property): Що саме ми змінюємо (колір, шрифт, відступ).
- Значення (Value): Яким саме воно має бути (червоним, 16px, жирним).

Приклад коду:
```css
h1 {
  color: blue;       /* робимо заголовок синім */
  font-size: 20px;   /* встановлюємо розмір шрифту */
}
```
